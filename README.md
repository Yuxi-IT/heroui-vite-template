# Bliss
- 一个由 [HeroUI 3](https://github.com/heroui-inc/heroui) + [FramerMotion](https://motion.net.cn/docs/framer) + [VITE](https://cn.vitejs.dev/) 构成的基础应用脚手架，帮助开发者更快地上手开发。

- An application scaffolding composed of [HeroUI 3](https://github.com/heroui-inc/heroui) + [FramerMotion](https://motion.net.cn/docs/framer) + [VITE](https://cn.vitejs.dev/) to help developers get started faster.
